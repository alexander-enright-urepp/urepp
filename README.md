# UREPP

A Sports Technology Recruiting Platform for baseball players to get discovered by college coaches.

Live at: https://urepp.vercel.app

## What It Does

UREPP connects baseball players with college recruiters through professional athlete profiles.

**For Players:**
- Create a professional recruiting profile
- Showcase stats, position, graduation year, location
- Update your profile anytime
- Get discovered by college coaches

**For Recruiters:**
- Search athletes by name, position, graduation year, or location
- Browse detailed player profiles
- Find the right talent for your program

## Features

- ✅ **Player Profiles** — Create and manage your recruiting profile
- ✅ **Profile Search** — Discover players by filters (position, year, location)
- ✅ **Dashboard** — Manage your profile and view activity
- ✅ **Authentication** — Secure login/signup with email
- 🔜 **Video Highlights** — Upload and showcase game footage (coming soon)
- 🔜 **Direct Messaging** — Connect players and recruiters (coming soon)

## Tech Stack

- **Frontend**: Next.js (App Router), React, TypeScript
- **Styling**: Tailwind CSS with custom baby blue theme
- **Backend**: Supabase (Auth, PostgreSQL Database)
- **Icons**: Lucide React
- **Deployment**: Vercel
